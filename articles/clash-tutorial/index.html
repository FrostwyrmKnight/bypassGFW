1️⃣ Clash 是什么？（工具 vs 燃料）

在开始之前，我们必须搞懂一个概念，这能帮您理解 90% 的问题：

概念 说明 类比
Clash Verge 这是一个"工具"（客户端软件）。它本身不提供翻墙能力。 超级跑车（没有油）
订阅链接 这是"燃料"（节点服务）。您需要付费购买。它包含服务器节点信息。 高级汽油

💡 核心概念：本教程教您如何使用工具（Clash），但您需要从服务商那里获取燃料（订阅链接）。

4️⃣ Windows 教程 下载与安装 Clash Verge

由于原版 Clash for Windows 已停止更新，Clash Verge 是目前公认的最佳替代品。

步骤 1：访问 GitHub 下载

Clash Verge 的官方下载地址在 GitHub 上。

  • 官方发布页： https://github.com/Meta-Repo/clash-meta/releases
  • ⚠️ 注意：GitHub 在国内可能访问缓慢，您可能需要一个临时梯子来下载。

步骤 2：选择正确的版本

在最新的 Release 页面，找到 Assets 部分，下载适用于 Windows 的版本：

您的电脑类型 下载文件 说明
64 位（推荐） Clash.Verge_x.x.x_x64-setup.msi 绝大多数新电脑都是 64 位
32 位 Clash.Verge_x.x.x_x86-setup.msi 较老的电脑

步骤 3：安装

下载后，双击 .msi 文件。和安装普通软件一样，一路点击"下一步"即可完成。安装完成后，Clash Verge 会自动运行，并在您电脑右下角的"任务栏"中显示一个图标。

5️⃣ 导入"订阅链接"（给跑车加油）

这是最关键的一步。您需要从您的服务商那里获得"订阅链接"。

步骤 1：复制订阅链接

登录您的服务商网站，找到"复制订阅地址"的按钮，点击它。订阅链接通常看起来像：

https://subscribe.example.com/api/v1/xxx?token=abcdef123456

步骤 2：导入到 Clash Verge

  1. 打开 Clash Verge 的主界面（双击任务栏图标）
  2. 点击左侧菜单栏的 "Profiles" (订阅配置)
  3. 在顶部的输入框中，粘贴您的订阅链接
  4. 点击右侧的 "Import" (导入) 按钮

✅ 导入成功后，您会在下方看到一个卡片，显示订阅名称和节点数量。Clash Verge 会自动下载节点信息。

6️⃣ macOS 教程：Clash Verge 与 ClashX

步骤 1：下载客户端

  • Clash Verge (macOS)：下载最新 .dmg 或 .zip 包（Apple Silicon/Intel 对应版本）。
  • ClashX：在官方发布页下载 ClashX.dmg。

步骤 2：安装与首次运行

  1. 安装：将 App 拖入“应用程序”文件夹，若出现“来自不受信任开发者”提示，前往“系统设置 → 隐私与安全”允许。
  2. 首次启动：授予网络权限；菜单栏会出现图标（闪电/小猫图标）。

步骤 3：导入订阅与更新

  1. 打开主界面，进入 Profiles（或“配置”）。
  2. 粘贴服务商提供的 订阅链接，点击 Import / Update。

步骤 4：系统代理与模式

  • 打开系统代理：Settings → 勾选 System Proxy。
  • 选择模式：Proxies → Mode: Rule（推荐）。

常见问题（macOS）

  • 无法启用系统代理：检查“网络”设置是否被 MDM 管理；重启 App 或更换网络位置。
  • Safari 仍直连：清理代理缓存，确认 Wi‑Fi/以太网服务都启用系统代理。

7️⃣ Android 教程：Clash Meta for Android

步骤 1：下载安装

  • 下载安装 Clash Meta for Android (ClashMeta) 的 .apk（ARM64 常见）。如遇“未知来源”提示，开启安装权限。

步骤 2：导入订阅

  1. 打开 App → Profiles/订阅 → 添加 URL 订阅。
  2. 点击 更新，拉取节点。

步骤 3：开启服务与分应用代理

  • 启动：首页点击 Start 启用服务，如需始终可用，开启“保持后台”。
  • 分应用代理：进入 Rules/Bypass 或 Per-App，给国内 App 设为直连，国际 App 走代理。

常见问题（Android）

  • 后台被杀：在系统“电池优化”中允许后台运行，锁定任务卡片。
  • 某些 App 仍直连：检查分应用代理列表及规则优先级。

8️⃣ iOS/iPadOS 教程：Stash 或 Shadowrocket

步骤 1：App 获取

  • Stash：在 App Store 购买并安装。
  • Shadowrocket：部分地区商店上架（付费），安装后可用。

步骤 2：导入订阅与启用

  1. 打开 App → Subscriptions/配置 → 添加 URL。
  2. 选择 Rule 模式，首次连接会提示添加 VPN 配置，点击允许。

常见问题（iOS）

  • 连接频繁断开：关闭“低数据模式”，检查蜂窝数据与后台允许。
  • 无法添加配置：在“设置 → 通用 → VPN 与设备管理”中手动删除旧配置后重试。

9️⃣ Linux 教程：Clash Verge 或命令行

方案 A：Clash Verge（桌面环境）

  1. 下载 AppImage 或对应包，赋予执行权限：chmod +x Clash.Verge.AppImage。
  2. 运行后在 Profiles 导入订阅，Settings 中启用系统代理（GNOME/KDE 通常生效）。

方案 B：命令行（服务器/轻量环境）

  1. 安装 clash-meta 核心，下载订阅后生成 config.yaml。
  2. 启动：clash-meta -f ~/.config/clash-meta/config.yaml，默认本地端口 127.0.0.1:7890。
  3. 为终端走代理：在 Shell 中设置环境变量 HTTP_PROXY 与 HTTPS_PROXY。
# 终端代理示例 (bash/zsh)
export HTTP_PROXY=http://127.0.0.1:7890
export HTTPS_PROXY=http://127.0.0.1:7890
# 验证
curl https://www.google.com -I

常见问题（Linux）

  • 系统代理不生效：检查桌面环境的网络代理设置；必要时以 iptables 或 systemd 服务方式处理。
  • 仅终端有效：桌面浏览器可能未读取系统代理，手动在浏览器内设置为 127.0.0.1:7890。

🔟 代理模式对比

模式 说明 速度 流量 推荐
Rule（规则） 智能分流。国内网站直连，国外网站走代理 ⚡⚡⚡ 💰 省 ✅ 强烈推荐
Global（全局） 所有流量走代理，包括国内网站 ⚡ 💸 多 ⚠️ 测试用
Direct（直连） 关闭代理，所有流量直连 ⚡⚡⚡ ✅ 无 ❌ 无法翻墙

1️⃣1️⃣ 开启代理，畅游网络！

步骤 1：开启"系统代理"

  1. 点击左侧菜单栏的 "Settings" (设置)
  2. 找到 "System Proxy" (系统代理) 选项
  3. 点击开关，使其变为绿色/开启状态

步骤 2：选择代理模式

  • 在左侧菜单栏点击 "Proxies" (代理)
  • 在页面顶部的 "Mode" 下拉菜单中，选择 "Rule"（推荐）

步骤 3：选择节点

在 Proxies 页面，您会看到一个 "Proxy" 卡片。对于新手：

  • 直接选择"自动选择"或"负载均衡" → Clash 会自动选择最快的节点
  • 或手动选择特定地区（香港、日本、美国等）

步骤 4：测试连接

打开浏览器，访问 https://www.google.com 或 https://www.youtube.com

✅ 能打开 = 成功！您已经掌握了 Clash！

⚠️ 常见问题解决

问题 原因 解决方案
导入后没有节点 订阅链接过期或网络问题 重新复制最新链接重新导入
网速很慢 节点距离远/负载高 切换到其他节点或选择"自动选择"
某些网站无法访问 规则模式可能被绕过 临时切换到 Global 模式测试
系统代理不生效 防火墙或权限问题 以管理员身份运行 Clash Verge

📊 安全建议

  1. 使用正规服务商 - 避免免费、不知名的代理服务
  2. 定期更新 - 及时更新 Clash 和系统
  3. 保护链接 - 不要公开分享您的订阅链接
  4. 监控流量 - 定期检查代理流量消耗
